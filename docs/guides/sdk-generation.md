# SDK Generation Guide

`dev/generate-sdk.sh` scans `src/**/*Result.php` classes, extracts their public constructor parameters, and emits TypeScript interface definitions to `packages/client-sdk/types.ts`.

## Usage

```bash
./dev/generate-sdk.sh
# or with custom paths:
./dev/generate-sdk.sh src packages/client-sdk/types.ts
```

The script is idempotent — re-run after every Result class change.

## How it works

1. Finds all files matching `src/*/Features/*/Application/*Result.php`.
2. Parses `public function __construct(...)` via regex — no autoloading required.
3. Maps PHP types to TypeScript equivalents (see table below).
4. Emits one `interface` per Result class into `packages/client-sdk/types.ts`.

### PHP → TypeScript type map

| PHP | TypeScript |
|---|---|
| `string` | `string` |
| `int`, `float` | `number` |
| `bool` | `boolean` |
| `array` | `unknown[]` |
| `\DateTimeImmutable` | `string` (ISO 8601) |
| `?T` (nullable) | `T \| null` |

## Output example

Given `src/Context/Billing/Features/ProcessPayment/Application/ProcessPaymentResult.php`:

```php
final readonly class ProcessPaymentResult
{
    public function __construct(
        public string            $paymentId,
        public int               $amountCents,
        public string            $currency,
        public \DateTimeImmutable $processedAt,
    ) {}
}
```

The generator emits:

```typescript
// Auto-generated by dev/generate-sdk.sh — do not edit manually.

export interface ProcessPaymentResult {
  paymentId: string;
  amountCents: number;
  currency: string;
  processedAt: string; // DateTimeImmutable → ISO 8601
}
```

## Extending the generator

The generator is a single PHP heredoc inside `dev/generate-sdk.sh`. Common extension points:

- **Add a type mapping** — extend `$typeMap` inside the heredoc.
- **Change the output format** — replace the `interface` template with `type`, JSON Schema, or OpenAPI component format.
- **Include non-Result classes** — change the glob pattern at the top of the PHP section.

## CI integration

Add to `Makefile`:

```makefile
sdk: ## Regenerate TypeScript SDK types
	./dev/generate-sdk.sh

sdk-check: ## Fail if generated types are out of date
	./dev/generate-sdk.sh
	git diff --exit-code packages/client-sdk/types.ts
```

Then add `make sdk-check` to the CI pipeline alongside `make ci`.
